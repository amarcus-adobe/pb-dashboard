APIs available for pulling police shootings and police brutality

/shootings will return all records in our DB

/shootings/{id} will return just record number id

Future functionality

/shootings?startIndex=0&size=20 will return the first 20 records

/shootings?startIndex=-1&size=20 will return the last (most recent) 20 records

/brutality will return all records in our DB

/brutality/{id} will return just record id

/brutality?startIndex=0&size=20 will return 20 records starting from the beginning

/brutality?startIndex=-1&size=20 will return the last (most recent) 20 records
